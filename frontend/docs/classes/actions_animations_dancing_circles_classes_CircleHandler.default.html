Class default

Represents a handler for managing circle animations.

Remarks

This class is responsible for creating and managing circles with various properties such as radius, position, and color. It provides methods for linear interpolation of these properties to create smooth animations.

Example

const circleHandler = new CircleHandler(0);
circleHandler.lerpColor();
circleHandler.lerpPosition(true);
circleHandler.lerpRadius();

lerpRadius

  • lerpRadius(): void

    Linearly interpolates the current radius towards the target radius. This method updates the currentRadius property by moving it a fraction of the way towards the targetRadius based on a fixed interpolation factor.

    Returns void

    Remarks

    The interpolation factor is set to 0.2, meaning the currentRadius will move 20% of the way towards the targetRadius each time this method is called.
