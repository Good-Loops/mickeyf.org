      ): PitchHueCommitterOutput

      Steps the committer for one frame.

      Behavior summary:

      • The returned pitchHueWeight01 attacks/decays smoothly toward 1/0 based on whether music is present and strong enough.
      • While enabled, a candidate pitch class must remain stable for stableThresholdMs before the committed hue updates.
