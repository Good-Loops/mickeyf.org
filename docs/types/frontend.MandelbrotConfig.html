    Type Alias MandelbrotConfig

    Public configuration surface for the Mandelbrot animation.

    Callers apply partial updates via Mandelbrot.updateConfig(patch) semantics (a shallow merge patch). Effective behavior is owned by the consumer (Mandelbrot.ts / MandelbrotTour), and some fields may be ignored or applied only at specific lifecycle points.
